Comparison between Part A and Part B

Part A: Custom CSS Implementation

Part A involved creating a responsive web page using custom CSS, media queries, and viewport. It allowed for full creative control over the design and layout. Learn more about Part A

Part B: Bootstrap Framework

Part B utilized the Bootstrap framework to build the web page. Bootstrap offers pre-designed components and a grid system for rapid development. Learn more about Part B

Challenges Encountered

Both implementations had their own challenges. Part A required more custom coding and design work, which could be time-consuming. Part B, while easier to use, might limit design flexibility.

Preferences

Your preference may depend on your project's goals. If you seek complete design control and have more time, Part A might be your choice. If you want a faster solution with a consistent look, Part B is a good option.
